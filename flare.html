Flare

Follow these steps for the Flare setup:

1. New location - Flare located at far end of most pad sights.

2. Install 3/4" conduit from flare disconnect to location of flare VFD panel, where VFD is located.

3. Install a QUAT box on conduit and continue conduit run to the end of pipe rack that runs right at flare stack, add flex from QUAT to panel.

4. Pull wire through conduit to QUAT box with enough wire to go.

5. Install 3/4" flex from QUAT box to SPCC panel or other type of flare VFD panel.

6. Install 3/4" flex from end of conduit run to flare blower motor.

7. Pull wire (#10 thn, brown/orange/yellow/green) from flare disconnect on panel rack to QUAT box, flare panel, VFD panel.

8. Pull wire from flare panel to flare blower motor and terminate in panel.

9. Terminating wires - Flare panel wires (white) coming from disconnect will terminate on P side of breaker inside VFD.

10. Blower motor wires will terminate on the drive inside panel. Will have to remove covers to access. (L1-brown as indicated on nameplate).
